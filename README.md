# how to run ?

- `npm install`
- `npm run dev`会监听文件变化并编译，编译完会执行代码。好处是直接运行代码看结果  
或者
- `npm run watch`会监听文件变化并编译，然后 f5 在 vscode 里运行。好处是方便断点调试

## 注意

> 在 vscode里测试时，会有时候结果不对，不知为啥。还是运行 dev吧。
