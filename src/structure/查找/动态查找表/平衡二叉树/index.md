# 平衡二叉树

平衡二叉树并不就是AVL树，它就是一个平衡的二叉树而已，AVL只是一种构建这种树的算法，红黑树也是种平衡二叉树呀.平衡二叉树通常是“almost”平衡

AVL树：
1. 对于任一节点，其左右子树的高度差的绝对值不超过1
2. 任意一棵子树都是AVL树

> AVL树这个名字是以发明它的两个作者的名字简写而来，不是平常的英文缩写
